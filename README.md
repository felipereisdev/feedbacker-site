# Feedbacker (Site)

A Vue.js 3 app with composition api for feedbacks management.


## Screenshots

![App Screenshot](https://github.com/felipereisdev/feedbacker-site/blob/master/screenshots/index.png)

  
## Installation

Install feedbacker-site with npm

```bash
  cd feedbacker-site
  npm install
  npm run serve
```
